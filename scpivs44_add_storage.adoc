= 스토리지 추가
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


. VMware vSphere 웹 클라이언트의 왼쪽 탐색 창에서 * Storage Systems * 를 클릭한 다음 을 클릭합니다 image:scpivs44_image6.png["오류: 그래픽 이미지가 없습니다"] * 추가 *.
. 스토리지 시스템 추가 대화 상자에서 기본 SVM 또는 클러스터 정보를 입력한 다음 * 추가 * 를 클릭합니다.
